## 🏡 Ejecutar localmente

1. Clona el repositorio
```shell
git clone https://github.com/ribaban-DAW/DynSoul.git
```

2. Entra al directorio
```shell
cd DynSoul
```

3. Instala las dependencias
```shell
npm install
```

4. Inicia el servidor
```shell
npm run dev
```

5. Abre http://localhost:4321 en tu navegador
